# Memory

Это классическая карточная игра, направленная на развитие памяти. Суть игры очень
простая. На столе «рубашкой» вверх выкладывается набор карточек. Далее игрок должен
выбрать две карточки, если картинки на карточках одинаковые, то карточки убираются со стола
или оставляют рисунков вверх, в противном случаи их переворачивают обратно.
